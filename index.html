HTML

  • HTML is the frame of the website, and the "elements" make up the content that is shown.
  • 'Head' contains all the meta data for the website, and helps render the page correctly
    • 'title' element defines the title for the webpage on the tab in the browser.
    • 'link' element connects an external resource to hte HTML document.
    • 'meta' element defines meta data, such as character sets, descriptions, keywords, author, and viewports.
    • 'script' element connects any client side scripts to the HTML file.
    • Note: anything in the head is not visable
  • 'Body' contains the elements shown to the user, and are what we want them to interact with.
    • 'body' is an example of semantic HTML, which has the tag directly explain what the element does.
    • 'h1->h6' elements represent the level of heading a given text block represents
      • Note: h1 is the highest, h6 is the lowest.
    • 'p' element represents a free block of text.
    • 'ul' element represents an unordered list.
    • 'ol' element represents an ordered list.
    • 'li' is often used inside lists to make bullet points.
    • 'img' element contains infor about pictures that are displayed.
      • Note: Contains various attributes, but 'src' defines the locaiton of the image.
    • 'br' element creates a line of empty space, or a page break.
    • 'a' element (anchor) creates links to the same, or other webpages.
      • Note: 'href' attribute points to the URL for the link.
    • 'header' element and all children appear at the top of the page.
    • 'footer' element and all children appear at the bottom of the page.
    • 'section' element adds all children to a section of code.
      • Note: Sections can be altered by using its 'id' and 'class' attributes in CSS stylesheet.
  • '!DECLARATION html' is a declaration that tells the browser what kind of document it should expect when opened.
    • Note: This is usually put at the very top of the HTML file.
