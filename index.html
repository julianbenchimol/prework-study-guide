HTML

  • HTML is the frame of the website, and the "elements" make up the content that is shown.

    • <head> Section:

    • <head> contains all the meta data for the website, and helps render the page correctly
    • <title> element defines the title for the webpage on the tab in the browser.
    • <link> element connects an external resource to hte HTML document.
    • <meta> element defines meta data, such as character sets, descriptions, keywords, author, and viewports.
    • <script> element connects any client side scripts to the HTML file.
    • Note: Anything in the head is not visable

    <body> Section:

    • <body' contains the elements shown to the user, and are what we want them to interact with.
    • <body> is an example of semantic HTML, which has the tag directly explain what the element does.
    • <h1->h6> elements represent the level of heading a given text block represents
      • Note: <h1> is the highest, <h6> is the lowest.
    • <p> element represents a free block of text.
    • <ul> element represents an unordered list.
    • <ol> element represents an ordered list.
    • <li> is often used inside lists to make bullet points.
    • <img> element contains infor about pictures that are displayed.
      • Note: Contains various attributes, but <src> defines the locaiton of the image.
    • <br> element creates a line of empty space, or a page break.
    • <a> element (anchor) creates links to the same, or other webpages.
      • Note: <href> attribute points to the URL for the link.
    • <header> element and all children appear at the top of the page.
    • <footer> element and all children appear at the bottom of the page.
    • <section> element adds all children to a section of code.
      • Note: Sections can be altered by using its <id> and <class> attributes in CSS stylesheet.
  • <!DECLARATION html> is a declaration that tells the browser what kind of document it should expect when opened.
    • Note: This is usually put at the very top of the HTML file.

CSS

  • CSS is used to define how webpage content is presented.
  • There are three ways to style a webpage using CSS:

      • 1. Inline CSS:

      • Written directly in the HTML file
      • Applies to a specific HTML tag, using a style attribute to style a specific page element only.
      • Is a quick and permanent change.
      • Is a less flexible change than style sheets because each inline must be edited individually.

      2. Internal CSS Stylesheet:

      • Embedded in <head> section.
      • Can configure CSS classes and IDs.
      • one change applies to all the tagged items.
      • Any rules are only applied to the individual page.

      3. External CSS Stylesheet:

      • Allows the ability to keep all CSS rules seperate.
      • Makes design changes easier.
      • Linked to HTML using <link> placed in <head>
        • Example: <link rel = "stylesheet" href = "./assets/style.css">

    Declarations:

    • Declarations contain two important components:
      • 1. CSS property we want to change.
      • 2. The new value of the property
        • Example: div{color: blue; text-align: justify}
        • div: the selector of a certain sector.
        • color, text-align: the property to be changed.
        • blue, justify: the new value of the property.

    Types of CSS Properties:

    • margin: How much space we want OUTSIDE an element. A margin of 0 indicates there are no extra space.
    • padding: Adds space around the content INSIDE the element.
      • Note: Padding is located inside the element, whereas margin is located outside.
    • When applying something to all elements on a page, use the * sector.
    • It is possible to link sectors if they have similar styles.
      • Example: header,footer{} means anything inside will apply to both header and footer elements.
    • <h1...h6> elements can be styled as sectors, along with <img> to style pictures, and <ul> to style lists.

    Linking a Stylesheet to the HTML Document:

    • In the <head> section:
      • <link rel = "stylesheet" href = "./assets/style.css">
        • Link creates a connection to the linked document.
        • Rel is the relationship between the current HTML document and the linked document.
        • href specifies the location of the external source. As it is in the assets folder, it is linked by "./assets/style.css"

    Adding Style using Class Attributes:

    • To use a class selector, use a period before hand.
      • Example: .card will select any class with the '.card' class assigned.
    • Any HTML element assigned to the class will gain the properties specified.

JavaScript

  • JavaScript is the tool that makes a static page interactive.
  • Variables are declared using "var" followed by the variable name.
    • Note: Variables aren't defined by type like in C#, but more like python.
  • To declare a function:
    • function FunctionName()
  • Linking a JS file to the HTML index:
    • <script src = "./assets/script.js">
  • To declare an array:
    • var arrayName = [content]
  • Arrays are not limited to a specific data type, and can contain different types.
  • For loops are a way to repeat certain code under specific conditions.
    • Delcare a for loop:
    • for(var x = 0; x < shapes.length; x++)
      • var x = 0 is the declaration of the local variable being used.
      • x < shapes.length is the condition for the loop to continue.
      • x++ is what is done if the conditions aren't met.
      • For this example, local variable 'x' will continue to add 1 while it is less than the length of the shapes array.
